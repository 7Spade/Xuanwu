- Install deps: npm install
- Start dev server: npm run dev
- Build: npm run build
- Start production server: npm run start
- Lint: npm run lint
- Lint with fixes: npm run lint:fix
- Type check: npm run typecheck
- Unit tests: npm run test
- Full local check: npm run check
- Genkit dev: npm run genkit:dev
- Genkit watch: npm run genkit:watch
- Firebase deploy app hosting: npm run deploy:apphosting
- Firebase deploy functions: npm run deploy:functions
- Firebase deploy firestore rules: npm run deploy:firestore-rules
- Windows utility commands: Get-ChildItem, Get-Content, Select-String, Set-Location, git status, git diff
